--- About
It's application for assign humanoid robots to a suitable job. Made using by HTML+CSS, JavaScript, Web SQL Database.
Browser supported:
- Chrome 4+
- Safari 4+
- Opera 11+
- iOS Safari 5+
- Android 3+
- Opera Mobile 11+
